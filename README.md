# Pronunciation Master

발음 명료도 평가 웹 애플리케이션

## 🌟 특징

- **20문장 완전 평가**: 기초 명료도, 음소 구분, 억양 리듬 종합 평가
- **S1~S10 레벨 시스템**: 과학적 평가 기준에 따른 명확한 레벨
- **상세 음소 분석**: L/R, P/F, TH 등 개별 음소별 정확도 분석
- **맞춤형 피드백**: 강점, 개선점, 연습 방법 제안
- **웹 기반**: 별도 설치 없이 브라우저에서 실행

## 🚀 시작하기

### 로컬 실행
1. 모든 파일을 같은 폴더에 저장
2. `index.html`을 웹 브라우저로 열기
3. 마이크 접근 권한 허용

### GitHub Pages 배포
1. 이 repository를 fork하거나 clone
2. Settings → Pages → Source에서 main branch 선택
3. Save 클릭
4. 잠시 후 `https://[사용자명].github.io/[저장소명]`에서 접속 가능
